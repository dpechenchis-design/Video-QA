# 🎬 EGA Video QA — Веб-додаток

Веб-додаток для QA-перевірки відео перед здачею клієнту.
CP відкриває сторінку → drag & drop відео або вставляє посилання → отримує звіт.

---

## Задеплоїти на Vercel (10 хвилин)

### 1. Anthropic API ключ
- Зайди на [console.anthropic.com](https://console.anthropic.com)
- Settings → API Keys → Create Key
- Скопіюй ключ (`sk-ant-api03-...`)

### 2. Завантаж на GitHub
- Створи новий **приватний** репозиторій
- Завантаж всі файли з цієї папки

### 3. Задеплой на Vercel
1. [vercel.com](https://vercel.com) → New Project
2. Підключи GitHub репо
3. **Environment Variables** → додай:

| Назва | Значення |
|-------|----------|
| `ANTHROPIC_API_KEY` | твій ключ Anthropic |

4. Deploy → отримаєш URL типу `https://ega-video-qa.vercel.app`

### 4. Готово
Відправ посилання команді — більше нічого встановлювати не треба.

---

## Як використовувати (для CP)

1. Відкрий посилання у браузері
2. Вибери вкладку **"Файл"** або **"Посилання"**
3. Drag & drop відео або вставте Frame.io / Drive URL
4. Натисни **"Запустити QA"**
5. Чекай 2-5 хвилин → читай звіт
